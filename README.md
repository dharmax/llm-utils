# @dharmax/llm-utils

A generic and sophisticated code-to-llm bridge including routing, prompt management, and session context.

## Features
- **Asker**: High-level interface for multi-provider LLM requests.
- **Model Router**: Dynamic scoring of models based on task capabilities.
- **LLMSession**: Continuity tracking with context management.
- **Prompt Engine**: Manifest-based templating and placeholder injection.
